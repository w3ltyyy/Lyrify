**Lyrify** — это высокопроизводительное расширение для **Spotify** (через **Spicetify**), созданное для меломанов, которые ценят точность и эстетику. Оно предлагает полную экосистему для просмотра, записи и модерации синхронизированных текстов песен.

### ✨ Ключевые возможности

- **Синхронизация с точностью до пикселя**: Плавная прокрутка текста с субмиллисекундной точностью.
- **Высокопроизводительный Recording HUD**: Специализированный интерфейс для ручной синхронизации. Записывайте таймкоды с помощью клавиатуры в режиме реального времени.
- **Современный дизайн**:
  - Интегрированные эффекты **Glassmorphism** и динамические живые фоны.
  - **Иммерсивный Полноэкранный Режим**: Чистый, минималистичный сплит-интерфейс с обложкой, парящими кнопками управления и кинематографичным прогресс-баром.
  - Полностью адаптивная типографика (размер шрифта, межстрочный интервал, максимальная ширина).
  - Динамические акцентные цвета, извлеченные из обложки трека.
- **Надежная архитектура**:
  - **Бэкенд**: API на Node.js на базе **SQLite** с использованием Write-Ahead Logging (WAL).
  - **Контрибьютинг**: Встроенная система отправки текстов для обновлений силами сообщества.
- **Умные альтернативы**: Автоматический поиск в **LRCLIB**, **Spotify API** или нативный DOM-скрейпинг.

### 🚀 Руководство по установке

#### 1. Сервис Бэкенда
Центральный узел для хранения данных и модерации.

1. Установите зависимости:
   ```bash
   cd backend
   npm install
   ```
2. Запустите сервис:
   ```bash
   node server.js
   ```
   *По умолчанию API работает на порту `8080`.*

#### 2. Развертывание расширения
1. Убедитесь, что [Spicetify-cli](https://github.com/spicetify/cli) настроен.
2. Перейдите в папку расширения: `cd spicetify-lyrics-sync`.
3. Запустите автоматический скрипт развертывания:
   ```bash
   ./scripts/build-and-install-spicetify-extension.sh
   ```
4. Активируйте расширение:
   ```bash
   spicetify apply
   ```

### 🎧 Инструкция по синхронизации (Recording Mode)
Если вы нашли песню без текста:
1. Откройте окно **Lyrify** и перейдите в **Settings** (Настройки).
2. Вкладка **Sync** -> Включите **Recording Mode**.
3. Наверху появится **HUD** (информационная панель).
4. Во время игры песни нажимайте **`[Space]`** или **`[Enter]`** в момент начала новой строки.
5. Нажмите **`[V]`** или иконку галочки, чтобы отправить текст на сервер.

### ⚙️ Конфигурация и кастомизация
Lyrify позволяет проводить глубокую настройку через панель параметров:
- **Общие**: Размер текста (22px-52px), Размытие фона, Яркость.
- **Внешний вид**: Межстрочный интервал, Ширина колонки, Затенение неактивных строк.
- **Синхронизация**: Переключатель режима записи.
